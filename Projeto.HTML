O que são Tags

Através de qualquer editor de texto, como Sublime Text, o NotePad++ ou até mesmo o bloco de notas, é possível criar um documendo com a extensão (.HTML) que será renderizado pelos navegadores

Conforme explicado anteriormente, este documento consiste em uma série de tags. As tags são códigos que definem toda a estrutura da página, tais como o seu tamanho, a fonte da letra, as cores ,as quebras de linha e etc. A marioria dos elementos do documento HTML são compostos por uma estrutura de abertura e uma de fechamento.

Há também tags de estrutura única, como a tag (Br) realizada uma quebra de linha

Estrutura básica do HTML

HTML é composto pelas tags (Cabeça=Head) e (Corpo=Body), sendo Cabeça tudo que será carregado antes de exibir para o usuário, como por exemplo o título da aba que ele está, o Corpo é tudo que está sendo mostrado dentro do site.

Falando sobre Tags

Temos tags que servem para deixar seu site mais bonito, modificando a formatação o tamanho e a cor das letras.

  • Como exemplo temos as tags (i) e (strong), sendo (i) para deixar seu texto em Itálico, e (strong) para deixar seu texto em Negrito

Outros dois exemplos de tags são (input) e (img). (input), (img) colocando o link de uma de uma imagen ou gift, ira aparecer o conteudo do link em seu site.

Atributos das Tags

existem tags com atributos mais unicos, que são utilizados por elas. Usando como exemplo a tag (input), ela tem tag chamada (type="text") e outra (type="number"). Que criam caixas para serem preenchidas em formulários. Caixas estas que o usiario pode colocar seu nome, idade, endereço etc..., estes são chamados Atributos das Tags.

Textos

Para escrever um texto, seja para trabalho, escola ou até mesmo um livro, você precisa de um título. Mesma coisa no HTML, Para você criar um título utiliza-se as tags (h1, h2, h3 até h6), sendo (h1) título principal, (h2) subtítulo, até o (h6).

Já para escrever um parágrafo usa-se a tag (p).

  • Como exemplo, vou usar o anime (One Piece), que é dividido em Arcos e Sagas.
  • Diferença entre Arco e Saga, enquanto a Saga é o todo de determinada parte de uma história, o Arcos são as partes que compõem esta saga, funcionando como uma espécie de subtítulo para o título principal, ou capítulos de um livro.

  • (h1) One Piece (/h1)

  • (h2) Saga East Blue (/h2)

  • (h3)Arco Romance Dawn (/h3)

  • (h3)Arco Orange town (/h3)

  • (h3)Arco Vila Syrup (/h3)

  • (h3)Arco Baratie (/h3)

  • (h3)Arco Arlong Park (/h3)

  • (h3)Arco Bando do Buggy: Após a Batalha! (/h3)

  • (h3)Arco Arco Loguetown (/h3)

  • (h3)Arco Arco Dragão Milenar (/h3)

Citei acima apenas uma saga do anime. Começando pelo Título (h1), Subtítulo (h2, h3).
